Тег Зачем нужен Атрибуты
h1..h9 Тег заголовка, размер завист от числа поле h. 1 - самый большой Style - стиль, align - вырвнивание
p Параграф Id- имя стилевого индексатора, style -стиль, hidden- скрывает содержимое от просмотра
table Таблица widtd- ширина таблицы
strong, i, em, small, del, sup Делает тект полужирным и указывает на важность, выставляет курсивом текст, выставляет текст курсивом и указывает на важность, перечеркивает текст, для создание текста в верхнем регистре widtd- ширина таблицы
a создание ссылок, указывает браузеру что данный текст является ссылкой href - адрес на который следует перейти, download -предлагает скачать данный файл
img нужен для того, чтобы отобразить на странице изображение alt - альтернативное имя картинки, size - задать размер изображения, widtd - ширина изображения
